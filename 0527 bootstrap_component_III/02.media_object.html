巢狀

媒體物件可以無限做巢狀，但是我們建議你別作過頭。將巢狀的媒體.media放在父媒體對象的.media-body內。

對齊

媒體物件中的媒體可以在你的 .media-body 內容的上部（預設）、中部、或下部與 flexbox 工具對齊。

媒體列表

由於媒體物件具有較少的結構要求，你也可以在清單 HTML 元素上使用這些 class。在你的ul或ol上，增加 .list-unstyled，以移除任何瀏覽器預設清單樣式，然後對你的li應用 .media。當然，你也可以使用間隔調整工具。
