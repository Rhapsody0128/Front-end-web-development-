CSS-Doodle

CSS-Doodle使用CSS grid的特性，可以在一個畫布上為每個單元格設計出隨機且複雜的數學運算

選擇器

:doodle(容器)

  1. :doodle{}內，為容器的布局樣式
  2. :doodle{}外，為每個單元格子的樣式
  3. 以下為繪製5x5/500px的grid，gap為格子間的距離
:doodle { @grid:5x5/500px; gap:10px; } background:#FFA

:container

:doodle { } :container{ @grid:5x5/500px; gap:10px; } background:#777; border:3px solid #000; transform:rotate(@rand(360deg)); @nth(odd){ background:#F33; } @even{ border-radius:2rem }

@at(row,col)

指定第幾列/第幾欄的單元格

:doodle{ @grid:5x5/500px; gap:10px; } background:#FFA; @at(4,3){ background:green; }

@random([ ratio ])

隨機選擇單元格，@random沒有比率時可以省略括號

使用比率，ratio比率介於0~1之間，預設值為0.5

:doodle{ @grid:20x20/500px; } @random { border-top: 1px solid #60569e; } @random { border-left: 1px solid #60569e; } @random(.2) { :after { content: ''; background: hsl(@rand(360), 60%, 70%); @size: @rand(10px); } }

@row(n,...)

選擇網格中第n列

:doodle{ @grid:5x5/500px; gap:10px; } background:#FFA; @row(even){ background:green; }

@col(n,...)

選擇網格中第n欄

:doodle{ @grid:5x5/500px; gap:10px; } background:#FFA; @col(odd){ background:green; }

屬性

@place-cell

相當於:doodle容器來定位單元格。第一個數值為水平，第二個數值為垂直，可以使用left、right、center、top、bottom或是絕對數值px...等等，就想是使用了position:absolute去調整位置

:doodle{ @grid:5x5/500px; gap:10px; background:#eee; margin:auto; } background:#FFA; @size:3rem; @nth(1){ @place-cell:0 top; background:red; z-index:1; } @nth(2){ @place-cell:25%; background:green; z-index:1; }

@shape

產生一個剪裁形狀，屬性質請參照官網

:doodle{ @grid:5x5/500px; gap:10px; background:#eee; margin:auto; @shape:heart; } background:#FFA; @shape:star;
