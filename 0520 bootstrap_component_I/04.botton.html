按鈕(button)

1.Bootstrap包括多個預先預定的.btn以及其他可以定義顏色的綴詞class

2.btn設計給button標籤使用 但也可以和a.input標籤一起使用

Link

3.btn-outline-* 可使用外框樣式

4. .btn-lg、.btn-sm使用較大或較小的按鈕，.btn-block使用區塊級按鈕

5. 啟用狀態與停用狀態

  • 為按鈕增加.active可以切換按壓啟用狀態
  • disabled屬性可以使用在button標籤上
  • a標籤沒有disabled屬性，可增加.disable class
