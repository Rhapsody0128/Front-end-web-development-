輸入欄位組(input group)

在 input 的任一側放置一個附加元件或按鈕。 您也可以在 input 的兩面都放置一個。另外

  1. input-group-prepend:放在前面的位置(第一個範例)
  2. input-group-append:放在後面的位置(第二個範例)
  3. 或兩邊各放置一個(第四個範例)
  4. label要放在表單群組外(第三個範例)
